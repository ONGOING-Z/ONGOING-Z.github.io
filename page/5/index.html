换入新ubuntu要做的几件事情

Posted on 2020-04-04 In Ubuntu

记录自己换入新ubuntu系统后的一些操作。

换源

到阿里云或者其他镜像网站复制对应安装版本的源

Install vim 8+

安装vim的方式

  • 将源文件下载下来后可以采用编译的方法进行。
  • 在命令行中安装

安装完毕后安装vim-plug，其作用是管理vim的插件。安装有以下两种方式

  • 在终端中键入下边命令
  • 将plug.vim下载下来后移入~/.vim/autoload文件夹下即可使用。

注: 使用vim-plug必须先安装git，不然不能使用!
使用自己的.vimrc

Install tmux

使用自己的.tmux.conf

Git

Install googlepinyin

在text entry setting可以设置，可能出现安装之后重新启动又得再次选择输入法的情况．

  • 在命令行中输入fcitx，检查其是否已经安装，如果未安装，执行sudo apt install fcitx进行安装。
  • 在命令行中执行im-config(Input Method Configuration)，进行一番点击之后在选择面板中选择fcitx
  • 安装googlepinyin，在命令行中执行sudo apt install fcitx-googlepinyin
  • 重启电脑，使配置生效。(注意: 不重启有可能在接下来的步骤看不到googlepinyin的选项哦！)
  • 在命令行中执行fcitx-config-gtk3，打开配置面板:
    • 点击+
    • 不要勾选Only Show Current Language，搜索一下就可以看到googlepinin了，加入即可。

注: googlepinyin对于ubuntu20也同样适用。

Install chrome

  • CLI安装
  • 到Chrome官网直接下载.deb包进行安装
    手动安装: sudo dpkg -i xxx.deb

卸载不必要软件

当然如果在安装过程中选择了最小安装，就直接跳过此步，因为那种最小安装方式只会安装浏览器和一些必要组件。

卸载内置视频软件totem，没有太多功能，使用smplayer代替

Install xdm

xdm是一款下载软件，使用过后发现下载速度相对浏览器下载会比较快。
这个链接是一位博主的，xdm官网在国外，下载很慢，这里下载会比较快。

  • 使用xdm在chrome官网下载.deb包
  • 使用xdm在vmware官网下载.deb包

Display netspeed

启动之后就可以在屏幕右上角出现网速了。
右键右上角图标，勾选run on startup，使其开机启动。

系统负载指示器

没啥用，是图形化的，不能显示数据

不那么鲜艳的绿色。

安装命令行打字软件gtypist

之前一直是在typing.com上来进行训练的，之后在一个帖子中突然发现了下方这个命令行软件，这里记录一下。

Curl 命令的使用

2023/02/08

前言: 临时没有postman做接口测试，就直接在 terminal 里做的测试。

发起Get请求

例如本地起的服务为"0.0.0.0 =8080/ask"，那么如何请求呢？

curl -G -d "name=mike" -d "height=170"  0.0.0.0 =8080/ask

这里-G不带的话，就为 post 请求了。

  • 带header参数
curl localhost =8080/api/rmi.do -H "key1:test1" -H "key2:test2"

测试网址链接

curl -I 你的网址

Claude Code

2026/05/04

最近因为 cursor 额度用完了，需要找个替代品。但是我想着另起一个账号再订阅一次的，刷短视频看到好多博主在用 claude cli + deepseek v4 pro。 我想着也尝试了。

很久之前都听说过 claude code，自己也尝试过 cursor cli，试用了下，发现并不是很惊艳，还是用回了自己的 gui 界面

但是这次使用 cc cli + v4 pro 模型，这个体验可以说很好、很好。

一个是关于配置的，一些规范可以集成到我维护很久的 dotfiles 仓库中去，但是 cursor 属于那种用完了，就用完了，我也没心思去把这个工具的配置留下来。 但是 cc 的用完的，我就马上维护起来了。

之前的 cursor，这个月的额度 5 天就给消耗完了。用的属实是很快。

那天晚上我抱着尝试下 v4 pro 的想法，让模型去找下我当前的项目有什么值得优化的，给了我几条建议，我执行完毕之后，程序性能提升了好多。 让我对这个模型更有信心了。

cc 出来有一两年了，但是我现在才开始使用它，想着应该早点使用的，应该可以给我节省好多钱，cursor 之前有几个月额度大半都没有去用。

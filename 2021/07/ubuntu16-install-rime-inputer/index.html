Ubuntu16 安装 Rime 输入法

2021/07/29

前言：之前一直使用的是谷歌拼音，但是奈何谷歌拼音的词库太老了，有时候很常见的一个词语我就得慢慢挨个选字。现在我的系统里安装的都是 Rime 输入法，这个输入法自己可以定制很多信息。谷歌拼音用的太久了，也应该换个更加省力的输入法了。

  • Rime 官网
  • 机器环境：Ubuntu16.04

这里我选用的是 fcitx-rime。

主要安装步骤如下：

  1. 在命令行中输入

    sudo apt install fcitx-rime

    安装完成后系统需要logout下，重新登录电脑

  2. 在命令行输入fcitx-config-gtk3打开配置面板，搜索 rime

Rime 的配置文件位于 = ~/.config/fcitx/rime

刚刚使用可能输出的字都是繁体的，需要打开 rime 配置界面：ctrl + grove, 这里的 grave 键 是指键盘左上角带有波浪线的那个键位。

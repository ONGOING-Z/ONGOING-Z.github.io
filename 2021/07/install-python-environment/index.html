Python环境安装

2021/07/27

前言：在 Linux 下关于 Python3.9 版本的环境安装，在此记录下。

首先在 Python官网 下载对应版本的源文件。

下载下来之后直接解压到文件夹 /usr/local/ 下

➜ pwd               
/usr/local/Python-3.9.6 # 路径

接下来对 Python 源码进行构建：

在 Python-3.9.6 的根目录下，执行

./configure
make
make install

注意：这里可能会出现权限不足的情况，直接 sudo 重试下。

这一步完成后，退出当前 bash/zsh 后重启执行以下命令查看 Python3 的版本

➜ python3 -V                        
Python 3.9.6

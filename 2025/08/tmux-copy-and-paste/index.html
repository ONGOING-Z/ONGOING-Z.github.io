Tmux 的复制和粘贴

2025/08/04

一、背景

在终端中使用 tmux 界面时，更快捷的复制粘贴设置。

二、按键步骤

  1. 按下 tmux prefix 键位 (默认是 ctrl + b)
  2. [: 进入选择界面，可以使用 vim 的上下左右和其他跳转键位
  3. 使用空格键触发选择
  4. y: 复制到系统剪切板中

三、配置

（一）安装

brew install reattach-to-user-namespace

（二）~/.tmux.conf中添加配置

bind -T copy-mode-vi y send-keys -X copy-pipe "reattach-to-user-namespace pbcopy"  # macOS

适配器模式

2025/07/24

一个转换层，起承上启下的作用。

翻译家也可以说是适配器。

适配器可以让不同的接口进行交互，充当两个不兼容接口之间的桥梁。

我想到收到请求，将之转换为 domain，返回响应，将 domain 转换为响应。

adapter wrap an object.

添加适配器前：外层直接访问 object，接口不同，访问不通

添加适配器后：可以直接访问 adapter，adapter 可以直接访问 object.

场景：装修的时候走水管时一堆的适配器

linux sed

2025/02/22

前言 = 有时需要在bash脚本中处理一些字符串的替换操作，发现 linux 下的 sed 命令可以实现这个功能。

很久以前就听过 linux 命令文本处理三剑客的大名，其中一个就是 sed ，使用了下发现还不错，模式匹配的格式就是通用的正则表达式。

例子： 将字符空格替换为-

echo "hello hhh" | sed 's/\ /-/'

macos直接修改源文件：

sed -i '' 'pattern' file.txt

生成一个备份文件

sed -i.backup 'pattern' file.txt

-E: 使用正则表达式。

一、什么是JavaBean

一种遵守编程规约的类:

  • 所有的属性都是私有的, 提供public的getter/setter方法
  • 必须有public的无参构造函数
  • 实现了Serializable接口

遵守了这些约定俗成规范的类就是JavaBean。

@Entity
public class Employee implements Serializable{

   @Id
   private int id;
   private String name;   
   private int salary;  

   public Employee() {}

   public Employee(String name, int salary) {
      this.name = name;
      this.salary = salary;
   }
   public int getId() {
      return id;
   }
   public void setId( int id ) {
      this.id = id;
   }
   public String getName() {
      return name;
   }
   public void setName( String name ) {
      this.name = name;
   }
   public int getSalary() {
      return salary;
   }
   public void setSalary( int salary ) {
      this.salary = salary;
   }
}

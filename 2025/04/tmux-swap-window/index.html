Tmux移动窗口

2025/04/08

一、背景

Tmux移动窗口方便重新编排我的工作空间。

二、一些操作

  1. 我想要互换窗口，例如将4窗口换到2窗口

    swap-window -s 4 -t 2

  2. 我想要保存当前会话，并且在使用的时候进行恢复

    • 保存：prefix + ctrl + s
    • 恢复：prefix + ctrl + r

  3. 关闭当前窗口：prefix + &

  4. 在macos上经常因为赋值粘贴导致当前的tmux windows死掉没有反应了

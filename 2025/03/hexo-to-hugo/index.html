博客移植之路：Hexo 到 Hugo

2025/03/29

一、背景

看到一个简洁的主题：

  • 主页界面会展示所有的博客列表，只限于标题展示
  • 不会进行分页，还需要让你点击上一页或者下一页，一页展示你所有的博客

二、移植过程

由于 hexo 和 hugo 主要是文章开头的 formatter 不一致，所以这边移植过程主要是把 hexo 的格式映射到 hugo 的格式。

使用的工具是 sed.

使用完后觉得是很方便的工具，搭配 deepseek 使用更加得心应手。

处理日期时间、替换关键字、处理图片、处理文件位置。

主要的思路就是把 hexo 各个关键字都替换成 hugo 的关键字，用 sed 的行替换就很方便，或者可以说我的替换就全部都是 sed 来完成的，搭配上 deepseek 给的表达式，就可以方便的完成了。

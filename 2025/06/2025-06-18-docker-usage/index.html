COPY 语法

  • 如果目标目录不存在，docker 会自动创建目录

例子：将目录docs.spring.io这个整个目录（包含目录及其内容）复制到app目录下

COPY docs.spring.io /app/docs.spring.io/

因为 COPY 时只会复制源目录里的内容，所以你需要指定目标目录，之后 docker 会自行创建。

查看占用体积

命令描述
docker system df显示占用的体积

前缀：docker image

命令描述
ls显示所有的 image 镜像文件
rm id删除对应 id 的镜像文件，注意不能有容器跑到这个镜像上

前缀：docker container

命令描述
prefix + ls显示所有的容器文件
prefix + stop id停止 id 容器
prefix + rm id删除 id 容器
docker start id启动已经存在的容器

端口映射

宿主机端口：容器内部端口

宿主机端口任意指定

容器内部端口必须为镜像指定的端口 (查看镜像信息可以获得这个信息)

退出 docker 进程 (macos)

在命令行中执行

osascript -e 'quit app "Docker"'

Vim markdown即时预览

2020/07/30

在vim中书写markdown文件挺方便的，但是有时侯需要预览一下效果，就有些不方便了，下边我找到一个插件，可以实现这个功能。

安装node.js

sudo add-apt-repository ppa =chris-lea/node.js
T sudo apt-get update+08 = '00'
sudo apt-get install nodejs

安装instant-markdown-d

sudo npm -g install instant-markdown-d

注 = 如果npm太慢，就改用cnpm进行安装。

npm install cnpm -g --registry=https://registry.npm.taobao.org

安装vim插件vim-instant-Markdown

可以搭配vim插件tabular和vim插件vim-markdown使用。如果没有安装这两个插件，在你的.vimrc文件中加入(注：这里我所使用的vim插件管理器是vim-plug)

Plug 'godlygeek/tabular' " 此插件必须在vim-markdown之前
Plug 'plasticboy/vim-markdown'

这两个插件安装完毕之后开始安装vim-instant-Markdown，在你的.vimrc插件部分添加

Plug 'suan/vim-instant-markdown'

执行=PlugInstall，插件安装完成后，在vim打开markdown文件就会在电脑默认的浏览器中打开一个窗口进行实时预览。

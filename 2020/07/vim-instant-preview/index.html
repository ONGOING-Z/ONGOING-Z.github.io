Vim markdown 即时预览

2020/07/30

在 vim 中书写 markdown 文件挺方便的，但是有时侯需要预览一下效果，就有些不方便了，下边我找到一个插件，可以实现这个功能。

安装 node.js

sudo add-apt-repository ppa =chris-lea/node.js
sudo apt-get update
sudo apt-get install nodejs

安装 instant-markdown-d

sudo npm -g install instant-markdown-d

注 = 如果npm太慢，就改用cnpm进行安装。

npm install cnpm -g --registry=https://registry.npm.taobao.org

安装 vim 插件 vim-instant-Markdown

可以搭配 vim 插件tabular和 vim 插件vim-markdown使用。如果没有安装这两个插件，在你的.vimrc文件中加入 (注：这里我所使用的 vim 插件管理器是vim-plug)

Plug 'godlygeek/tabular' " 此插件必须在 vim-markdown 之前
Plug 'plasticboy/vim-markdown'

这两个插件安装完毕之后开始安装vim-instant-Markdown，在你的.vimrc插件部分添加

Plug 'suan/vim-instant-markdown'

执行=PlugInstall，插件安装完成后，在 vim 打开 markdown 文件就会在电脑默认的浏览器中打开一个窗口进行实时预览。

双系统的安装：windows 和 ubuntu

2022/09/04

前言： 因为最近疫情的原因，可能需要在家办公。但是因为上次想从 ubuntu16 换到 ubuntu22 的原因，导致自己电脑上 windows 系统不能启动的原因，我需要重新装一下 windows 系统。

我的电脑

  • 两块硬盘
    • 第一块：120G 固态
    • 第二块：2T 机械 现在选择将 windows 装在 120 G 固态上，将 ubuntu 装在机械硬盘上。

Windows 安装

  • 提前准备好 PE U 盘，复制好 windows 镜像到 u 盘中
  • 使用 PE 的分区功能给硬盘分好区，可以快速分区，也可以自定义分区。
    • 因为只有 120G，所以分了三个区 (C = 40G, D: 20G, E: 52G)
  • 从 PE 中启动，右键镜像文件选择装载
  • 点击 setup.exe 文件

我这里点击了之后安装程序有报错：

我们无法创建新的分区，也找不到现有的分区。

最后使用复制方法安装的，复制解压后镜像中的四个文件

boot 文件夹
sources 文件夹
bootmgr 文件

到已经分好区的系统盘 (之前分好的 C 盘). 重启计算机即可。

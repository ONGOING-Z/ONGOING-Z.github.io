install windows and ubuntu with two disks

Posted on 2022-09-04 In Linux

前言:
因为最近疫情的原因，可能需要在家办公。但是因为上次想从ubuntu16换到ubuntu22的原因，导致自己电脑上windows系统不能启动的原因，我需要重新装一下windows系统。

我的电脑

  • 两块硬盘
    • 第一块：120G固态
    • 第二块：2T机械
      现在选择将 windows 装在 120 G 固态上，将 ubuntu 装在机械硬盘上。

Windows 安装

  • 提前准备好 PE U盘，复制好windows镜像到u盘中
  • 使用 PE 的分区功能给硬盘分好区，可以快速分区，也可以自定义分区。
    • 因为只有120G，所以分了三个区(C: 40G, D: 20G, E: 52G)
  • 从PE中启动，右键镜像文件选择装载
  • 点击 setup.exe 文件

我这里点击了之后安装程序有报错：

最后使用复制方法安装的，复制解压后镜像中的四个文件

到已经分好区的系统盘(之前分好的C盘).
重启计算机即可。
